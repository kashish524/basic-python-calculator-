# Scientific Calculator - Python Tkinter

Ye ek simple aur advanced Scientific Calculator hai jo Python Tkinter ka use karke banaya gaya hai. 
Isme basic operations ke sath scientific functions bhi add kiye gaye hain.

## Features
- Addition, Subtraction, Multiplication, Division
- Scientific Functions: sin, cos, tan, log, sqrt
- Dark/Light Theme Toggle
- Error Handling
- Clean and Simple User Interface

## Technologies Used
- Python
- Tkinter (GUI)
- Math Module

## How to Run
1. Python install hona chahiye.
2. Project folder me jao.
3. File run karo:
   python calculator.py
